## What is ciMate?

**ciMate** is an open-source CLI that manages the full lifecycle of your CI/CD pipelines — from initial setup to ongoing health monitoring and automated remediation.

```bash
npx cimate init        # Scaffold CI config for your project
npx cimate audit       # Detect issues in your pipeline
npx cimate heal        # Auto-fix common CI problems
```

## Packages

| Package | Description |
|---------|-------------|
| `@cimate/cli` | Command-line interface |
| `@cimate/core` | Detection engine & pipeline analysis |
| `@cimate/patterns` | 30+ CI failure patterns & remediation strategies |
